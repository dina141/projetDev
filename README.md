Description du projet:
C'est un mini site e-commerce développé avec HTML, CSS et Java Script.
Le site s'appelle "ART" et vend des imitations de tableaux celebres.

Caractéristiques du site:
- L'affichage des produits se fait dynamiquement avec JS (pas de multiples pages HTML).
- Le filtrage par catégorie est côté client.
- L'authentification (inscription/connexion) est simulée avec localStorage.
- Validation des formulaires avec les Regex.


Comment l'utiliser:
Ouvrez simplement le fichier index.html dans Google Chrome ou Firefox.

Structure des dossiers:
- index.html (page d'accueil)
- content/ (les pages html: inscription, connection, produits, commande)
- style/ (le fichier css)
- javascript/ (le fichier js)
- images/ (les photos des produits)
- readme.txt (ce fichier)
- la video utilisee pour l'accueil
